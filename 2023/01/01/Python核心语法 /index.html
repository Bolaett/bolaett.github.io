基本知识

注释

层次

代码缩进（tab）以及冒号( : )

文件编码声明

保留字

标识符（变量名）

数据类型

整形int
  • 十六进制 0x
  • 八进制 0o
  • 二进制 0b
  • 相互之间可以直接互相运算，不用转换，但是结果用十进制返回
浮点float
  • 所有的小数
复数complecated
  • a+bj的形式
字符串String
  • ‘abcdefg’ 单引号
  • “abcdefg” 双引号
  • ‘’’abcdefg’’’ 三引号
布尔Boole
  • True==1
  • False==0
  • 只有四种情况为假 ：1. False None； 2. 数值零；3. 空序列 ；4. 自定义对象实例

输入输出函数

输入

a=int(input("提示字符"))
input( )函数只接受字符串类型

输出

  • print用一次就自动换行一次
  • print(a,b,c) 中间空一格依次输出
  • print(xxx,end='') 不换行，如果字符串是空的就紧密拼接
  • print(r'xxxxxx') 取消转义，原样输出

类型转换

  • int( ) 只有十进制的字符串才能用
  • float( )
  • str( )
  • repr( )
  • eval( ) 计算字符串中的有效表达式
  • chr( ) 返回字符
  • ord( ) 返回ASCII码
  • hex( ) 转16进制
  • oct( ) 转8进制
  • bin( ) 转2进制
  • list( ) 转换成列表
  • tuple( ) 转换成元组

运算符

算数运算符

+ - * / % // **

  • % 不一定是正数，甚至不一定是整数，符号与第二个操作数相同
  • // 向下取整
  • int( ) 取整数部分
  • math.floor( ) 向下取整
  • math.ceil( ) 向上取整
赋值运算符 =

右边赋给左边

比较运算符

> < != ==

逻辑运算符

and or not

位运算符（二进制数不能用逻辑运算符）
  • << 左移
  • >> 右移
  • & 与
  • | 或
  • ~ 非
  • ^ 异或（异1同0）

优先级

幂 > 取反 正负 > 乘除 > 加减 > 移位 > 与 > 异或 > 或 > 比较

条件表达式

K= A if condition else B(真A假B)
C++中为 K= condition ? A : B(三目运算符)

流程控制

解释型语言，必须从上到下顺序执行
顺序、选择（分支）、循环

分支
  • if... 并列使用可以用作满足多个条件的判断
  • if...else... 补集
  • if...elif...elif...else... 当遇到第一个真的时候就跳出了，条件间无交集

C++中的switch：

循环
  • 计次循环（for） 条件循环（while）
  • range( )左闭右开，生成一系列整数，range对象
  • 可以用list( )把range对象变成列表
  • 遍历字符串for ch in string
  • break 与if 联用，跳出本层循环
  • continue 与if 联用，跳出本次循环
  • pass 占位

序列（数据结构）

  • 列表 元组 集合 字典 字符串 —>存储多个值的连续内存空间

  • 索引 0，1，2，3，4，…，n

  • 索引 -n , -(n-1) , … , -1

  • 切片sname[start:end:step] 左闭右开，冒号

  • 序列相加用加号，同类型才可以相加

  • 序列数乘 直接用*

  • 初始化指定长度列表 [None]*n

  • 检查是否是序列成员 in

  • 长度 len 最大值 max 最小值 min 求和 sum

  • enumerate( )组成索引序列

  • listname.append( )末尾追加

  • 删除序列 del

  • 通过索引或键访问元素或值

  • 仅能通过索引修改值

  • 删除单个元素仅能通过索引删除 del

  • seq.remove( ) 根据值删除，只删第一个，不存在则抛出异常

  • seq.pop( ) 根据索引删除

  • seq.count( ) 出现次数

  • seq.index( ) 首次出现的下标

  • seq.sort(key=None,reverse=False) 可以作为单独语句使用，改变原数组，key=str.lower不区分大小写

  • newseq=sorted(seq) 生成新列表，参数同前
